Консольная утилита для работы с дальномером по USB.

Проект PC-утилиты.
* Получение и сохранение калибровочных значений.
* Работа с логом устройства.
* Стирание памяти.
* Получение измерений в реальном времени.


Опции сборки:
-DDEVELOPER_BUILD=1
	Сборка утилиты в варианте "для разработчика".
-DGRAPH_SUPPORT=1
	Поддержка в утилите графического вывода параметров.


lascond.exe -loader -id 000000AAh -disk f
